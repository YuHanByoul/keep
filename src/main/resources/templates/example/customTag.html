커스텀 태그 속성 (kattr Attr) custom dialect in thymeleaf


kattr:select_code (get code data)

단일 selectBox 일때

< kattr:select_code="셀릭트박스 Id/name" grpCd="그룹코드" selectedCd="디폴트 코드 값 " upprCd="상위 코드값" onchange="커스텀 할 function" >

< span kattr:select_code="codeTagId" th:attr="grpcd=G0001" selectedCd="A0002" onchange="change">select Code select Code

< div kattr:select_code="codeTagId_1" grpCd="G0001" upprCd="A0001" selectedCd="B0001" >select Code
select Code

kattr:select_codes(get code data)

selectBox 그룹코드 값 depth 체크 후 갯 수만큼 selectbox 생성
< kattr:select_codes"셀릭트박스Id" >
(id는 차례대로 id_1 , id_2, id_3 등으로 셀릭트박스Id+'_순서' 로 생성)



kattr:check_code(get code data)

< kattr:check_code="체크박스 Id/name" >

< span kattr:check_code="checkId" grpCd="G0001" upprCd="B0001" selectedCd="C0001">

kattr:radio_yn (get code data)

< kattr:radio_yn="라디오버튼 Id/name">
(id는 차례대로 id1 , id2로 라디오Id+'순서' 로 생성, 라디오버튼 첫번째값 value는 'Y' 두번째값 value는 'N')

< span kattr:radio_yn="nlogin_perm_yn" label1="허용" label2="불가" defaultVal="N" >


kattr:date_pic

< kattr:date_pic="날짜input Id/name" dateDefault = "날짜 디폴트 값" 디폴트 값은 날자포맷과 동일한 포맷일 때 적용 됨 (선택)
dateForm="yy/mm/dd" 날자 포맷변환 디폴트 값은 yymmdd>
