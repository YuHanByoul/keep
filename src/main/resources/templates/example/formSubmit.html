form submit


<form th:action="@{/}" method="post">

form을 post방식으로 submit할때에는 th:action을 속성을 사용하고 속성의 값으로 @{/XXX} 형태를 사용하도록 한다.
이렇게 하는 이유는
thymeleaf에서 자동으로 form내부에 아래와 같은 csrf token input태그를 넣어주어 스프링 시큐리티를 통과할 수 있게한다.
< input type="hidden" name="_csrf" value="5f01c90e-942d-4282-9466-a7eb77c865cb" >
